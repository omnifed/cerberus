## Props

These props can be passed to the `AspectRatio` component.

| Prop        | Type         | Description                                    |
|-------------|--------------|------------------------------------------------|
| `ratio`     | `ConditionalValue<number>`     | The aspect ratio of the Box. Common values are: `21/9`, `16/9`, `9/16`, `4/3`, `1.85/1`      |
