## Contributing to Cerberus

<strong>Learn how to help improve Cerberus.</strong>

## How to Contribute

Cerberus is an open-source project, and we welcome contributions from the community. Whether you're a seasoned developer or just starting out, there are many ways you can help improve Cerberus.

1. **Report Issues**: If you find a bug or have a feature request, please open an issue on our [GitHub repository](https://github.com/omnifed/cerberus)
2. **Submit Pull Requests**: If you have a fix or a new feature, feel free to submit a pull request. Make sure to follow our [contribution guidelines](#contribution-guidelines).
3. **Documentation**: Help us improve our documentation by suggesting edits or adding new content.
4. **Spread the Word**: Share Cerberus with your friends and colleagues. The more people use it, the better it gets!

## Contribution Guidelines

You can find our contribution guidelines in the [CONTRIBUTING.md](https://github.com/omnifed/cerberus/blob/main/CONTRIBUTING.md) file in the root of the repository. Please read through it before submitting any contributions.

## Code of Conduct

We expect all contributors to adhere to our [Code of Conduct](https://github.com/omnifed/cerberus/blob/main/CODE_OF_CONDUCT.md) to ensure a welcoming and inclusive environment for everyone.
