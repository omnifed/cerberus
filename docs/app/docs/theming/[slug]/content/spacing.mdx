## Tokens

Cerberus provides the following spacing tokens:

`theme.tokens.spacing`

| Token    | Value | Visual                                                    |
| -------- | ----- | --------------------------------------------------------- |
|  none    |   0   |  <SpacingVisual value="none" />                           |
|  xs      |   0.25rem   |  <SpacingVisual value="xs" />                       |
|  sm      |   0.5rem   |  <SpacingVisual value="sm" />                        |
|  md      |   1rem   |  <SpacingVisual value="md" />                          |
|  lg      |   1.5rem   | <SpacingVisual value="lg" />                         |
|  xl      |   2rem   |  <SpacingVisual value="xl" />                          |
|  2xl     |   3rem   |  <SpacingVisual value="2xl" />                         |
|  3xl     |   4rem   |  <SpacingVisual value="3xl" />                         |
|  full    |   31.25rem | <SpacingVisual value="full" />                       |
