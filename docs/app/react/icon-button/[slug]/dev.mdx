## Sizes

The `IconButton` component comes in two sizes: `sm`, `lg`. You can set the size by using the `size` prop.

<CodePreview preview={<SizesDemo />}>
```tsx title="button.tsx"
import { IconButton } from '@cerberus/react'
import { Tuning } from '@carbon/icons-react'
import { HStack } from 'styled-system/jsx'

function SizesDemo() {
  return (
    <HStack>
      <IconButton size="sm" usage="filled">
        <Tuning />
      </IconButton>
      <IconButton size="lg" usage="filled">
        <Tuning />
      </IconButton>
    </HStack>
  )
}
```
</CodePreview>

## As Child

The `IconButton` component can be used as a child component. This is useful when you want to use the button styles on a different element.

<CodePreview preview={<AsChildDemo />}>
```tsx title="button-link.tsx"
function ButtonLink() {
  return (
    <IconButton asChild>
      <Link href="/react/icon-button">
        <Tuning />
      </Link>
    </IconButton>
  )
}
```
</CodePreview>

## Primitives

You can utilize the primitive components to customize the button.

| Component | Description |
| --------- | ----------- |
| IconButton    | The context provider for the Button parts |

<CodePreview preview={<CustomButtonPreview />}>
```tsx title="custom demo"
import { IconButton } from '@cerberus/react'
import { Tuning } from '@carbon/icons-react'
import { css } from 'styled-system/css'

function CustomButtonPreview() {
  return (
    <IconButton
      className={css({
        bgColor: 'danger.bg.initial',
        color: 'danger.text.initial',
        rounded: 'md',
        transform: 'skew(-10deg)',
        _hover: {
          bgColor: 'black',
          color: 'yellow',
        },
      })}
    >
      <Tuning />
    </IconButton>
  )
}
```
</CodePreview>

## API

### Props

The `IconButton` component is an abstraction of the primitives and accepts the following props:

| Name     | Default | Description                            |
| -------- | ------- | -------------------------------------- |
| palette     | `action`   | The color palette (any of the Cerberus palettes) of the button. |
| usage | `filled`    | The style treatment of the button.    |
| shape | `sharp`     | The shape of the button.              |
| asChild | | Render the Button as the child component. |
